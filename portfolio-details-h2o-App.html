H2Optimize

Project Repository

The project focused on building a smart water recycling recommendation system using machine learning. It featured both a web and mobile interface, where users could input water parameters like pH, TDS, and turbidity. The system predicted the water quality category using a stacked classification model and then offered customized reuse suggestions. The system served both home and industrial users via a mobile and web app, adapting recommendations based on user type. Hence, recommended appropriate reuse methods tailored to the category.

The dataset used for this project was sourced from Kaggle. It consisted of key water quality parameters such as pH, TDS (Total Dissolved Solids), and Turbidity. This data enabled us to train a stacked classification model to predict water quality based on just these three input values. Link

Many regions lack accessible tools to assess water reusability. Users whether households or industries often don’t know how to recycle water safely or where it can be reused. Manually testing water quality in laboratories is often costly and time-consuming, especially for basic assessments. The challenge was to create an easy, cost-effective, and accessible way to predict water quality.

We built a smart App in React Native and Python Flask backend with machine learning. A stacked classification model analyzed input parameters and categorized the water. Based on the category and user type (home or industrial), the system provided targeted reuse and recycling recommendations. The platform used REST APIs to connect with both a mobile and web interface, ensuring accessibility and ease of use.
