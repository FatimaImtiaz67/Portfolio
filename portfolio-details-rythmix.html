Rythmix - Habit and Class tracker

Project Repository

Rythmix is a full-stack web, developed using the MERN stack (MongoDB, Express.js, React, Node.js). A responsive web platform aimed at helping users manage personal habits and class schedules. It combines habit tracking, event scheduling, and academic calendar management in one seamless dashboard.

The main challenge was adapting the concept to a responsive web application while maintaining user-friendly design and rich interactivity. Key challenges included recurring event logic of both Habits and Classes, and syncing user activity with a calendar interface.

Implemented modular components for Authentication, Profile Management, Habit Scheduling, and Class/Event features using React and Bootstrap CSS. Integrated a calendar component to allow users to visualize and manage habits, classes, and events in monthly/weekly views. Also Habits and classes Creation/Join with Frequency, Color, and Icon Customization.

Key Features

  • Class Scheduling
  • Habit Creation
  • Habits Streak Maintenance
  • Calendar View with Event Filtering
  • Reminders for Classes, Habits
  • Account Settings Panel
