Retail Inventory Analytics -Excel & SQL

Project Repository

Built a data-driven inventory tracking solution using Excel and SQL to monitor weekly stock levels, filter trends by category, and support restocking decisions. Used PivotTables and slicers to make SKU performance easy to analyze, highlighting patterns in stockouts and reorder needs.

Used the following two simulated inventory datasets: Link 1

Link 2

Manually tracking inventory across multiple SKUs and categories often leads to inefficiencies, missed restocking windows, and poor demand forecasting. Without a structured reporting system, it’s difficult to quickly identify which products frequently go out of stock or need timely replenishment.

Imported raw stock and sales data into Excel, using SQL queries for structured extraction and filtering. Used PivotTables to group SKUs by category, week, and stock status. Integrated slicers for dynamic filtering by product type, region, and time period. Built summaries that forecast restocking needs based on stock levels and reorder thresholds.

Key Features

  • Weekly Inventory Overview
  • Out-of-Stock Alerts
  • PivotTables and Slicers
  • Performance View
  • Data Cleaning
